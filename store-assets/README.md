# Chrome Web Store 截图清单

商店页面至少上传一张清晰的功能截图。建议从已安装的扩展中截取并制作以下 3 张 1280×800（或 640×400）PNG：

1. **主界面**：展示 A 股、H 股、美股、国际指数标签及至少两条行情卡片。
2. **市场分类**：选中 H 股标签，只显示 `HK` 开头的自选股票。
3. **添加自选**：展示“添加股票”输入框及 `sh600519 / hk00700` 示例占位文字。

制作要求：

- 截图使用最新版本扩展，不要包含浏览器账号、书签、个人信息或与产品无关的界面。
- 保持清晰、无变形、全幅铺满画面；不要叠加夸张收益或“稳赚”等误导性文案。
- 以 `main.png`、`market-filter.png`、`add-stock.png` 命名后放入本目录，供商店上传。
