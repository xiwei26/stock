# Chrome 网上应用店资料

将以下内容复制到 Chrome Web Store Developer Dashboard。

## 商品详情

**名称**

股票行情助手

**简短说明**

快速查看 A 股、港股、美股与国际指数行情，支持管理自选股票。

**详细说明**

股票行情助手是一款轻量、便捷的行情查看扩展。点击浏览器工具栏图标，即可查看自己关注的证券最新报价与涨跌幅。

主要功能：

- 按 A 股、H 股、美股和国际指数分类查看自选行情
- 自定义添加和删除证券代码
- 自动刷新行情，并清晰展示最新价、涨跌额与涨跌幅
- 深色简洁界面，适合快速查看

使用说明：

1. 点击工具栏中的「股票行情助手」图标。
2. 点击「添加股票」，输入证券代码，例如 `sh600519`、`hk00700` 或 `usAAPL`。
3. 使用顶部标签按市场分类查看自选列表。

权限说明：

本扩展仅请求访问腾讯行情接口 `https://qt.gtimg.cn/*`，用于读取用户主动添加代码对应的公开行情数据。不会读取或修改用户浏览的网页内容。

数据与免责声明：

行情数据由第三方服务提供，可能存在延迟或误差。本扩展仅供信息参考，不构成任何投资建议、邀约或保证；投资决策请自行判断并承担风险。
